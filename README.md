# Get-Things-Done
A todo list app made with MongoDB, NodeJs, Express and EJS

## Requirements for Development

- `node`
- `npm`

## Setup for Development

- `git clone https://github.com/dhiraj161298/Get-Things-Done.git`
- `cd Get-Things-Done`
- `npm init`
- `node app`
- `open localhost:3001 in browser`
